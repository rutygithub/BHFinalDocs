傻瓜下载github 上面有关BH哥的评注的文件

MacOS

1.打开终端（terminal)

2. 终端（terminal) 打命令输入：     cd Desktop
（这个命令的意义是把你终端的路径转到桌面上）

3.终端（terminal) 打命令输入：      git clone https://github.com/rutygithub/BHFinaldocs.git
（这个命令的意义是克隆下载github路径里面的文件到桌面本地）




等待下载完成，就会发现BHFinalDocs的文件夹在你的MacOS的桌面上，然后你打开BHFinalDocs，逐一打印里面的文件和文件夹里的文件就可以了。
注意： 一定要用彩色打印，因为哥的评注都是用不同颜色的笔搞定的。
